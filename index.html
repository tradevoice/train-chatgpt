Re-Training ChatGPT On Pine Script

This project contains various Python scripts and files designed to download each chapter of the Pine Script Manual v5 from TradingView and aggregate them into a single text file. This aggregated file is particularly optimized for training a ChatGPT model.

Download Project Files

Project Files

Below are the project files included in the downloadable ZIP:

  • `download_chapters.py` - Automates the downloading of each chapter of the Pine Script Manual v5 from TradingView. Each chapter is saved as a separate PDF.
  • `combine_pdfs_to_text.py` - Takes all downloaded PDF chapters and merges them into a single text file for easy model training.
  • `PineScript_Manual_v5_Combined.txt` - A text file containing all the individual chapters combined. This file is optimized for training ChatGPT.
  • `PineScript_Manual_v5_Chapters` - A folder containing the individual PDF chapters of the Pine Script Manual v5, each named according to its chapter title.
